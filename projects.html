My Projects

Bird sound detection using MWSG spectrogram

The project is based on an idea to detect bird sounds in real field recordings. We proposed an unsupervised algorithm to enhance spectrogram which is generated using multiple window method. Later after filtering the spectrogram using Savitzky-Golay filter we captured bird specific sound cues using directional filters.

Image recognition system based on speech command

Designed an image recognition system based on the input given through speech. The goal of the project is to assist visual-impaired persons to know the coordinates of the object they are looking for just by their speech. The system on overall achieved an accuracy of 93.75%

ALS and Parkinsons diseases detection using voice as bio-marker

Built a speech classifier to detect Amyotrophic Lateral Sclerosis (ALS) & Parkinsons (PD) diseases based on voice as bio-marker and compared the disease detection across various devices. The best classification accuracy of 92.2% has been received on Zoom_H6 microphone recorded test data.
